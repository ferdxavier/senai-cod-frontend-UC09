# LH Games - Front-End Básico

Este projeto foi desenvolvido como parte da Unidade Curricular **UC09 (Codificação para Front-End)** do curso de Desenvolvimento Full-stack no **SENAI**. O objetivo principal foi criar uma interface responsiva para uma loja de jogos, aplicando conceitos básicos de HTML, CSS e integração com frameworks.

## 🚀 Funcionalidades

* **Página Inicial (Index):** Layout com Navbar, Carrossel de banners dinâmico e seção de produtos em destaque usando Cards.
* **Página de Login/Cadastro:** Formulários estruturados para autenticação de usuários e criação de novas contas.
* **Responsividade:** Uso de classes do Bootstrap para adaptação em diferentes tamanhos de tela.
* **Interatividade:** Botão "Voltar ao Topo" e manipulação básica de elementos com JavaScript/jQuery.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação das páginas.
* **CSS3:** Estilização personalizada (arquivo `estilo.css`).
* **Bootstrap 5:** Framework para agilizar o layout e componentes (Navbar, Carousel, Cards).
* **JavaScript & jQuery:** Scripts para comportamento da interface e validações simples.
